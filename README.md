# rota

Doctor rota application for Excel. Built to explore the basics of VBA and macros. 

## Instatllation

* Clone the repo.
* Open `rota-test.xlsm` in Excel.
* Enable editing and enable macros.

## Usage

Press the 'Go' button. 

Alternatively, run the macro through the Developer tab.
